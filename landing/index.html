⚠️ Mac showing "app is damaged"?

macOS blocks apps from unidentified developers. Here's the fix:

  1. Open the DMG and drag BuddyList to Applications
  2. Open Terminal (search in Spotlight)
  3. Run this command: sudo xattr -rd com.apple.quarantine /Applications/BuddyList.app
  4. Enter your Mac password when prompted
  5. Open BuddyList from Applications
This removes the quarantine flag macOS adds to downloaded files.
